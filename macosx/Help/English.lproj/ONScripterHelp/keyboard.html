Keyboard Shortcuts

NScripter games can use a simple "menu bar" to provide control over various functions, such as volume levels and automode. Since ONScripter does not (yet) support the menubar functionality, many of these options can be controlled by keyboard shortcuts. These keyboard shortcuts are described below.

Basic Controls

Space/Return — Advance Text

Both space and return will advance the text. The function of the return key is identical to a left click and will select buttons as well; the space bar will not select buttons.

Escape — Right-Click Menu

The escape key is identical to a right-click and will bring up the in-game menu.

F — Toggle Full Screen

Toggles in and out of full screen mode.

Text Display Controls

A — Automode

Enables/disables automode in titles that support it. While in automode, the text will automatically advance. Automode will be disabled if you use the mouse or keyboard to manually advance the text.

O — Page-at-Once Mode

Toggles page-at-once mode. In page-at-once mode, the text display is not animated and the entire block appears at once.

0, 1, 2, 3 — Text Display Speed

Adjusts the speed at which text appears, if the game script does not override it.

Sound Controls

M — Mute

Toggles audio mute.

V — Volume Adjustments

Starts interactive sound volume editing. The options are displayed in the title bar of the window. It is not recommended to use this option in full-screen mode.
